1 Introdução
As AACCs, chamadas comumente de atividades complementares,
caracterizam uma modalidade de atividade curricular, realizada pelo aluno, de
forma paralela às do curso, dentro ou fora do ambiente escolar, que contribui
para a implementação e enriquecimento do perfil do formando.
1.1 Sistema de atividades complementares
Ferramenta disponível para os coordenadores acadêmicos visando a
realização do controle das atividades complementares dos alunos através de
uma conexão segura utilizando a internet.1.3 Requisitos de hardware e software necessários
oMicrocomputador com conexão a internet;
oNavegador de internet (Internet Explorer, Mozilla Firefox, Opera, etc.): usar
sempre a versão mais atualizada.
Configurações necessárias no Mozilla Firefox:
oFerramentas – Opções – Conteúdos – Habilitar a opção “Permitir JavaScript”.
Configurações necessárias no Internet Explorer:
oExibir – Tamanho do texto – Selecionar a opção “Média”.
1.4 Normas gerais de utilização do sistema
1.4.1 Sobre cadastro/manutenção de atividades complementares
oSomente o coordenador poderá aprovar atividades complementares.
oOs coordenadores e os alunos poderão cadastrar, alterar, excluir e transferir
atividades dentro dos anos de vigência dos cursos em andamento.
oApenas as atividades ainda não aprovadas poderão ser alteradas e excluídas.
oNão é possível transferir atividades para alunos do 1º ano.
oNão é possível transferir atividades cuja carga horária tenha sido completada
para o ano corrente.
